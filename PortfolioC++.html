Terminal Game Engine [C++]

Sept 2024

An engine for creating 2D terminal-based games, built without any graphics libraries.

  • Framerate-independent simulation (delta-time based)
  • Efficient rendering (only the changing parts of the terminal are reprinted)
  • Particle system
  • Collision system (OnCollisionEnter/Exit)
  • Sorting layer system
  • Easy to add new levels / objects
  • Backgrounds support

Dining Philosophers Problem [C++, QT]

Jan 2025

Implementation of the classic Dining Philosophers Problem using C++ and the QT framework.

Two algorithms were implemented:

  • Hold and wait, demonstrates how deadlocks can occur when resources are held while waiting for others.
  • Chandy/Misra algorithm, prevents deadlocks by prioritizing resource allocation and managing ownership efficiently.
